# Saucedemo Automation Framework

Playwright + JavaScript based POM (Page Object Model) automation framework for Saucedemo.

## Tech Stack
- Playwright
- JavaScript
- Node.js

## Framework Structure

s360-automation/
- pages/ (Page Object classes)
- tests/ (Test files)
- utils/ (Helper functions)
- fixtures/ (Test data)
- playwright.config.js

## Features Covered
- Login (Positive, Negative, Locked user, Empty fields)
- Logout
- E2E Purchase Flow
- Sort Products
- Remove from Cart
- Cart Badge Count
- Cancel Checkout
- Product Detail Page
- Multiple Products Checkout

## How to Run

Install dependencies: npm install

Run all tests: npm test

View HTML Report: npx playwright show-report
